# Archive

Linux/X11-only configs preserved for reference. **Not deployed by `install.sh`.**

- **`i3/`** — i3 window manager config (X11). Won't run on macOS.
- **`compton/`** — compton compositor config (X11). Won't run on macOS.
- **`original.xkb`** — XKB keyboard layout dump (X11). Won't run on macOS.

If you ever set up a Linux box again, copy these into place manually as a starting point.
